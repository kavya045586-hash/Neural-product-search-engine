# Neural-product-search-engine
"An end-to-end recommendation system using a Two-Tower Neural Network architecture to provide real-time, semantic product search and discovery."

## ğŸ—ï¸ Technical Architecture
The core of this project is a **Two-Tower Neural Network** built with TensorFlow/Keras. 
- **Item Tower:** Processes categorical data (Brand, Category) and textual data (Product Titles) into a 64-dimensional embedding space.
- **User Tower:** Learns user preferences and maps them into the same shared vector space.
- **Inference:** Recommendations are generated by finding the nearest neighbors in the embedding space using cosine similarity.
